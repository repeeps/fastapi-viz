# FastAPI, 눈으로 보기

**요청 하나가 서버 안에서 흐르는 보이지 않는 길 — 라우팅·검증·의존성 주입·이벤트 루프·커넥션 풀을 코드가 아니라 움직이는 그림으로 읽는다.**

FastAPI(Python 비동기 웹 백엔드)의 **런타임 안쪽**을 **수식·코드 없이**, 요청 카드가 파이프라인을 통과하고 한 이벤트 루프가 여러 요청을 번갈아 처리하고 의존성 트리가 채워지고 커넥션 풀이 고갈되는 걸 직접 보며 익히는 인터랙티브 학습 자료입니다. 모든 흐름·타이밍·검증·QoS 판정은 **브라우저에서 실시간**으로 계산돼요. 문법 강좌가 아니라 "**왜 이렇게 도는가**"를 보는 눈을 기르는 게 목표예요. (로봇 노드 통신을 다루는 자매 트랙 "ROS2, 눈으로 보기"와 짝을 이뤄요 — 둘 다 "보이지 않는 런타임"을 시각화.)

## 시작하기

`index.html`을 브라우저에서 열면 끝이에요. 빌드도, 설치도 없어요. 처음이라면 **0번 입문**(요청 하나의 여정)부터 순서대로.

```
open index.html
```

## 구성 — 0번 입문 + 16챕터 (요청 여정 → async → 의존성 → 바깥 세계 → 운영)

**0번 · 서버가 하는 일** — 브라우저가 요청을 보내면 서버 안에서: ASGI 문지기 → 라우팅 → 검증 → 의존성 → 핸들러 → 응답. 전체 파이프라인과 두 하이라이트(async·DI)를 한눈에.

- **PART A · 요청의 여정** — 01 HTTP 요청·응답(메서드·상태코드) · 02 라우팅(경로/쿼리·매칭 순서)
- **PART B · 입력과 출력** — 03 Pydantic 검증(422 문지기) · 04 응답 모델(거름망)
- **PART C · async와 이벤트 루프 (하이라이트)** — 05 동기 vs 비동기 · 06 이벤트 루프(동시성≠병렬성) · 07 루프를 막지 마라(블로킹·threadpool) · 08 동시 요청과 백그라운드
- **PART D · 의존성 주입 (하이라이트)** — 09 Depends(선언하면 채워줌) · 10 중첩·캐싱(요청 단위) · 11 yield 생명주기(셋업·정리 보장)
- **PART E · 바깥 세계** — 12 DB 세션·커넥션 풀(쿼리 순간만 점유·고갈) · 13 N+1 쿼리(쿼리 폭발) · 14 인증(JWT=서명, 암호화 아님)
- **PART F · 실전·운영** — 15 미들웨어·에러 처리(양파 껍질) · 16 배포·워커(워커 vs async)+너머

각 챕터는 **직관 → 인터랙션 → 결론**의 4스텝으로, 슬라이더·버튼으로 직접 만지며 익혀요.

## 특징

- **수식·코드 없는 직관 입문** — 요청 카드가 파이프라인을 통과하고, async가 잘못 쓰여 이벤트 루프가 얼어붙고, 큐가 꽉 차 메시지가 버려지고, 위조된 JWT가 서명 불일치로 거부되는 걸 눈으로 봐요.
- **모든 시뮬은 실시간 계산** — 상태코드 분기, 이벤트 루프 스케줄(실행은 하나·대기만 겹침), 블로킹 시 직렬화, DI 트리 해석·캐싱, 커넥션 풀 고갈, N+1 쿼리 카운터, JWT 서명 검증, 미들웨어 양파, 워커 부하까지 전부 브라우저에서.
- **두 하이라이트** — 입문서가 잘 안 짚는 **async 이벤트 루프(C)**와 **의존성 주입(D)**에 가장 큰 비중. "왜 한 일꾼이 수천 요청을?", "async인데 왜 더 느리지?", "핸들러가 db를 어떻게 받지?"의 답을 끝까지.
- **정확성 가드레일** — 동시성≠병렬성(실행 막대 비겹침), "async≠빠름"(동기 블로킹은 def로 threadpool), DI 캐시는 요청 단위(전역 아님), JWT는 서명이지 암호화 아님 — 입문자가 가장 많이 틀리는 지점을 정조준.
- **의존성 0** — 순수 HTML+JS. 공통 엔진은 `shared/lib.js`(`VZ.AL` 스텝 플레이어), FastAPI 도메인 렌더러는 `shared/lib-api.js`(`VZ.API` 요청카드·파이프라인·검증·이벤트루프·DI트리·미들웨어양파·커넥션풀·JWT), 테마는 `shared/theme.css`.

## 구조

```
index.html              허브
00-intro.html           입문 (요청 하나의 여정)
01..16-*.html           16챕터
shared/lib.js           VZ.AL(스텝 플레이어) + 공통 유틸
shared/lib-api.js       VZ.API — FastAPI 도메인 렌더러
shared/theme.css        다크 테마
shared/favicon.svg
```

비영리 교육 목적으로 직접 만든 인터랙티브 학습 자료입니다.
